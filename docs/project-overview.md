# Project Overview

## Name

My Archive

## Vision

My Archive is a personal knowledge and archive management platform.

Its purpose is to centralize all saved information in one place instead of being scattered across messaging apps, screenshots, notes applications, browsers, bookmarks, and files.

## Purpose

The system allows users to create, organize, search, manage, and retrieve their personal archives through traditional CRUD interfaces and AI-powered tools.

## Core Concept

The platform manages multiple archive types.

Each archive type has:

* Dedicated pages
* Dedicated CRUD operations
* Dedicated fields
* Dedicated business rules

## Archive Types

## Artificial Intelligence

Artificial intelligence is an enhancement layer.

The application must remain fully functional without AI.

AI features include:

* Classification
* Tag suggestions
* Summaries
* Search assistance
* Knowledge retrieval
* Archive creation assistance

## Interfaces

The platform provides:

* Web Interface
* REST API

from the first release.

## User Roles

* Admin
* User

No complex permission system is required.
