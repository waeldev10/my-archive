# Archive Types

## Supported Types

* Note
* Link
* Article
* Image
* File
* Todo
* Plan
* Project
* Idea
* Bookmark
* Course
* Book
* Snippet
* Website
* Journal
* Prompt

## Rules

Each archive type:

* Has its own table
* Has its own model
* Has its own CRUD
* Has its own pages
* Can be searched
* Can be tagged

## Tags

Tags are global across the entire system.

Examples:

* Laravel
* AI
* Business
* Design
* Open Source

A single tag may be attached to any archive type.

## IDs

All archive tables must use ULID identifiers.
